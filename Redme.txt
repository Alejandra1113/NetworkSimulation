Este proyecto se hizo con dotnet6.0, por lo que es muy probable que no abra si se tiene dotnet5.0. Para abrirlo se puede usar el code o ejecutar directamente desde  "./Program/bin/Debug/net6.0/Program" (si va a usar este ultimo la consola debe tener la direccion del projecto, ya que los archivos que cargamos esta afuera en la raiz del proyecto). Tenemos implementado dos algoritmos de correccion de errores, si quisiera cambiar entre ambos debe escribir "ParityCheck" o "CheckSum" en config.txt
